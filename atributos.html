Atributos más comunes

1.- class: se usa para dar estilos a traves de CSS

2.- id: es un identificador, sirve para seleccionar un elemento ya sea desde javaScript, o para hacer navegación a través de anclas. El identificador debe ser único.

3.- title: es un atributo que ayuda acceder a cierto elementos mostrando una descripción del mismo elemento. Pasar el mouse encima del elemento, pero directamente en la pagina web.

4.- data-* : es un atributo que nos permite guardar algun valor en la etiqueta HTML.
